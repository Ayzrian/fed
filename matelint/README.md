# `@mate-academy/matelint` npm package

## Usage
```
 $ npm install matelint
```
next
```
 $ npx matelint
```
## Config
file name - `matelintrc`
```js
{
  'ignore': [
      "node_module",
      "**/test/*.html",
  ],
  'rules': {
    'rule-id': true,
  },
}
```
